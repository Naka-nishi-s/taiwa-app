# taiwa-app

対話型のひな形作成アプリ

## 使用方法

### 推奨

1. パッケージをインストールして実行します:

   ```shell
   npx taiwa-app
   ```

2. 1 の際に出てくる質問で、yes を選ぶと以下が実行されます。

   - `npm install hello-world-npm`
   - index.html の作成

### 非推奨

1. パッケージをインストールします:

   ```shell
   npm install -g taiwa-app
   ```

2. アプリケーションを実行します:

   ```shell
   taiwa-app
   ```

3. 2 の際に出てくる質問で、yes を選ぶと以下が実行されます。

   - `npm install hello-world-npm`
   - index.html の作成
